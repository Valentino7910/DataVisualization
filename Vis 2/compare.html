Comparison of data on deaths from COVID-19 in OECD countries

The interface at the top of this chart includes several action elements to adjust the displayed data:
Drop-down menus for country selection: Two drop-down menus are shown here to select data from two countries for comparison, respectively. The user can select a different OECD country from a drop-down menu and the chart will update the data based on the selection.
Year selection drop-down menu : The user can select the year of the data to view.
Trends this year: Clicking this button will display trends in deaths within the selected year.
All trends: Click this button to view all trend data from the beginning of the pandemic to the current day.
With these features, users can flexibly view and compare COVID-19 death data from different countries and at different times.
